# Humanized Application Executor Agent

## Steps
1. Login
2. Navigate job URL
3. Scroll page
4. Upload resume
5. Fill form fields
6. Paste cover letter
7. Simulate typing delays
8. Submit

## Anti-Detection Rules
- 50–150ms typing delay
- Random mouse paths
- 30–180 sec idle per action
- Max 15 applications/day/platform
